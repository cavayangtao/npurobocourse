《智能无人系统综合设计》以机器人操作系统（ROS）为软件平台，结合智能无人机、无人车等硬件设备，融合多学科知识内容，探索竞赛项目牵引、理论实践结合、重视团队合作的“人工智能+无人系统”教学模式。
